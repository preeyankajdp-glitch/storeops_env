Guide

What This Assistant Can Do

This assistant works on D-1 historical report data, not live inventory. It works best for clear quantity and variance questions.

Good Question Types

  • Store totals: one store and one item
  • Rankings: top stores or top variance items
  • Breakdowns: city-wise or date-over-date comparisons

How Answers Are Generated

  • Deterministic engine: used for the final table and result
  • Benchmark agent: used for the hackathon environment behind the scenes
  • User view: focused on simple business questions and evidence rows
