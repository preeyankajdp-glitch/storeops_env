What This Tool Can Do

This UI is built for D-1 analytics, not live inventory. It works best for structured quantity and variance questions.

Question Levels

  • Easy: single store, single item totals
  • Medium: variance ranking and city breakdowns
  • Hard: cross-store ranking and date-over-date change

Agent Modes

  • Heuristic: deterministic baseline for submission
  • LLM: optional OpenAI-compatible planner
  • Auto: tries LLM first, then falls back
